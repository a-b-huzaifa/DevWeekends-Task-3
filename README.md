# JS Fundamentals — Three Interactive Demos

This project is a premium, dark-themed interactive showcase demonstrating core JavaScript concepts like DOM manipulation, closures, promises, event handling, scope, and hoisting. 

It contains three hands-on frontend demos integrated into a single-page layout:

1. **Interactive Counter**: Demonstrates closures and scope with a responsive UI, state tracking, and a history log showing the last six actions.
2. **Filterable To-Do List**: Showcases arrays, object mapping, and DOM manipulation with features to add tasks, toggle completion states, filter by status (All, Active, Done), and clear completed items.
3. **Async JS Quiz**: Highlights asynchronous execution, timers, and promises (via `async/await`) with a quiz that loads questions mock-asynchronously, tracks scores in real-time, displays interactive correct/wrong feedback, and computes final performance grades.

**Tech Stack**: Modern Semantic HTML5, CSS3 Custom Properties (variables), and Vanilla JS.
